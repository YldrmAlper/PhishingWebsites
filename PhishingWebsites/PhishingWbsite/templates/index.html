URL Oltalama Nedir?

Oltalama (Phishing), siber suçluların meşru web sitelerini taklit ederek kullanıcıları kandırma girişimidir. Bu saldırılar genellikle kullanıcı adı, şifre, kredi kartı bilgileri gibi hassas bilgileri çalmayı amaçlar.

Nasıl Çalışır?

Sistemimiz, girilen URL'yi 30'dan fazla özellik açısından analiz eder ve makine öğrenimi algoritmaları kullanarak URL'nin meşru mu yoksa oltalama mı olduğunu tespit eder.

İncelenen Özellikler

URL Yapısı: IP adresi kullanımı, uzunluk, @ sembolü ve çift slash gibi yapısal özellikler
Alan Adı Bilgileri: Alt alan adı sayısı, alan adı yaşı, kayıt uzunluğu
Güvenlik Sertifikaları: SSL durumu, HTTPS kullanımı
Sayfa İçeriği: İframe kullanımı, harici bağlantılar, form hedefleri
